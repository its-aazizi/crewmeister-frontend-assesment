# Crewmeister Frontend Assesment

## Setup

- This project has two parts, the web and the api. The web is a react app that uses vite. The api is a node app that uses express.

## To start web

```bash
1. cd web/
2. rename ".env.example" to ".env"
3. yarn install
4. yarn vite
```

### To run tests on web
> yarn test
> yarn coverage

## To start api

```bash
1. cd api/
2. npm install
3. npm run dev
```
